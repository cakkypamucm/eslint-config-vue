This is `vue2+` `eslint` config

### Install

```shell
npm i --save-dev github:cakkypamucm/eslint-config-vue
```

### Use

Add `eslint-config-vue-cakkypamucm` to the "extends" array in your `eslint` config file

<pre>
{
    "extends": ["some-other-config-you-use", <b>"eslint-config-vue-cakkypamucm"</b>]
}
</pre>

### For synergy

Add [stylelint-config-vue-cakkypamucm](https://github.com/cakkypamucm/stylelint-config-vue) to your project
